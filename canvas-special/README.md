# 超多经典 canvas 实例

**普及**：`<canvas>` 元素用于在网页上绘制图形。这是一个 HTML 图形容器，必须使用脚本来绘制图形。

**注意**：IE 8 以及更早的版本不支持 `<canvas>` 元素。
